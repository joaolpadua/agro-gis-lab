# Agro GIS Lab

Projeto de estudo em análise geoespacial aplicada ao agronegócio.

## Funcionalidades

- Leitura de shapefile (IBGE)
- Cálculo de área municipal
- Consumo de API SIDRA (produção de soja)
- Tratamento de dados
- Exportação para CSV

## Tecnologias

- Python 3.11
- GeoPandas
- Rasterio
- Pandas
- Requests
- Git

## Objetivo

Construir base sólida para análise agrícola espacial e futuramente trabalhar com NDVI e imagens Sentinel.
